# 画布设计

完整手册：`read_file(".reasonix/skills/anthropic/canvas-design/SKILL.md")`

两阶段流程：
1. **设计哲学创作** (.md) — 4-6 段的视觉美学宣言
2. **画布实现** (.pdf 或 .png) — 高度视觉化、极少文字

## 核心原则

- 空间 > 文字：思想通过形式、色彩、构图传达
- 设计师级工艺：每处留白、每个元素都精心打磨
- 字体创意使用：`canvas-fonts/` 目录中的字体
- 输出：单页 PDF/PNG + 设计哲学 .md

## 注意

- 禁止模仿已有艺术家作品
- 文字极简，作为视觉元素而非段落
